# TodoApp

React + TypeScript で作成したシンプルな Todo リスト Web アプリです。  
タスクの追加・編集・削除や、優先度や期限管理、完了タスクの表示切替などが可能です。

Link: https://kz91.github.io/todo

---

## 特徴

- **タスク管理機能**
    - タスクの追加、編集、削除が可能
    - 完了済みタスクはチェックボックスで管理
    - タスクの優先度（高・中・低）を設定可能
    - タスクの期限を設定可能

- **便利な機能**
    - 未完了タスクのみの表示切替
    - 期限切れのタスクを赤文字で強調表示
    - 優先度ごとに色分けされたバッジ表示
    - 編集・削除時に確認ダイアログを表示

- **永続化**
    - LocalStorage に保存されるため、ブラウザを閉じてもタスクが保持される

- **操作性**
    - Enter キーでタスク追加・保存、Esc キーでキャンセル
    - IME 変換中の Enter 入力は無効化

---

## 使用方法

1. タスクを追加する
    - タスク名を入力
    - 優先度を選択
    - 期限を入力（省略時は1週間後が自動設定）
    - 「追加」ボタンをクリックまたは Enter キー

2. タスクを完了済みにする
    - チェックボックスをクリックして完了／未完了を切替

3. タスクを編集する
    - 「編集」ボタンをクリック
    - タスク名、優先度、期限を変更
    - 「保存」ボタンをクリックまたは Enter キー

4. タスクを削除する
    - 「削除」ボタンをクリック
    - 確認ダイアログで「削除する」を選択

5. 完了済みタスクの表示切替
    - チェックボックスで完了済みタスクの表示を ON/OFF

## ファイル構成（主要ファイル）

- `src/App.tsx` : アプリのメインコンポーネント
- `src/TodoList.tsx` : タスク一覧と編集・削除ダイアログの管理
- `src/EditTodoDialog.tsx` : タスク編集用ダイアログ
- `src/DeleteTodoDialog.tsx` : タスク削除確認ダイアログ
- `src/todoStorage.ts` : LocalStorage への保存・読み込み処理
- `src/types.ts` : Todo 型定義
